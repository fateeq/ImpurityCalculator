About

The USP Ractopamine Impurities Calculator was created by me, Farah, a former chemist delving into the tech world. The calculator is a beginner JavaScript application created for the purposes of my portfolio.

The inspiration for the calculator came from a problem I faced often at work when I was a chemist. The application is essentially a clean-looking, error-free alternative to the spreadsheet calculations my colleagues had to perform and that I had to review.

While I drew inspiration from problems at work, this application was created solely for my practice.

Tools used to build the calculator: HTML, CSS, vanilla JavaScript
